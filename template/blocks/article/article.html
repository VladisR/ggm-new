Blask поделился аналитикой по iGaming-рынкам за 13–19 апреля

{% set badgeAvatar = 'img/autor-badge.jpg' %} {% set badgeName = 'Ксения Булыгина' %} {% include "blocks/author-badge/author-badge.html" %}

Сервис подвел итоги прошедшей недели на iGaming-рынках по всему миру. Ниже — топ-5 стран по росту и падению Blask Index.

Взлеты недели:

  • Сомали: +16,9;
  • Лаос: +14,3;
  • Сальвадор: +13,5%;
  • Испания: +11,8%;
  • Камбоджа: +11,4%.

Падения недели:

  1. Ирландия: −23,3%;
  2. Чад: −21%;
  3. Швейцария: −18,8%;
  4. Южная Корея: −16,6%;
  5. Либерия: −16,4%.
«В этот раз рост стал более фрагментированным. Часть рынков двигалась из-за спорта, часть — из-за календарных эффектов, не связанных напрямую с гемблингом», — поделилась команда Blask.

72% всех рекламных нарушений в Индии связаны с офшорными букмекерами

{% set badgeAvatar = 'img/autor-badge-2.jpg' %} {% set badgeName = 'Михаил Видрашко' %} {% include "blocks/author-badge/author-badge.html" %}

Совет по рекламным стандартам Индии (ASCI) сообщил, что в 2025–2026 финансовом году офшорные букмекерские платформы стали крупнейшим источником нарушений рекламного законодательства. На их долю пришлось 72,14% всех рекламных объявлений, признанных несоответствующими нормам.

Это произошло несмотря на вступивший в силу в августе 2025 года закон PROGA, который запретил онлайн-игры на реальные деньги и любую их рекламу.

Основной площадкой нарушений остается интернет:

  • 97,3% всех выявленных нарушений пришлись на цифровые медиа;
  • 82% цифровых нарушений были размещены через спонсируемую рекламу в социальных сетях.
  • Сальвадор: +13,5%;
  • Испания: +11,8%;
  • Камбоджа: +11,4%.

Дополнительную обеспокоенность у властей вызывает инфлюенсер-маркетинг. Из 1 609 проверенных рекламных интеграций почти 97% потребовали корректировки из-за нарушений правил раскрытия рекламы. Более половины таких случаев были связаны с ограниченными категориями, включая нелегальный гемблинг.

Всего за отчетный период ASCI рассмотрел 11 581 дело — на 21% больше, чем годом ранее.
